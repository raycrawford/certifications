# Overview
Azure Search seems a lot like hosted Solr.

Data can be pushed to Search via the REST API or .NET SDK.  

Data can be pulled to Search via indexers which currently include:
- BLob storage
- Table storage
- Azure Cosmos DB
- Azure SQL
- mSQL on Azure VMs
